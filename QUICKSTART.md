# MindPalace 快速上手指南

## 5 分钟快速体验

### 1. 安装依赖

```bash
pip install -e .
```

### 2. 配置 API（首次使用）

```bash
python -m src
```

选择 `⚙️ 设置` → `配置 API / 模型`，输入你的 API Key。保存后可以立即检测是否可用。

### 3. 开始使用

```bash
python -m src
```

默认 Scout 内容源现在偏向人文社科。如果你想临时切到混合或科技预设，可以直接运行：

```bash
python -m src scout --preset mixed
python -m src scout --preset tech
```

## 推荐使用流程

### 第一次使用

1. **配置 API** → 选择 `⚙️ 设置`
2. **抓取内容** → 选择 `📚 文章库` → `🎯 发现新文章`
3. **浏览文章** → 选择 `📚 文章库` → `📚 浏览文章`
   - 从列表中选择感兴趣的文章
   - 选择操作：
     - 📖 生成导读精炼版（快速了解）
     - 🌐 查看原文（浏览器打开）
     - 🏛️ 发起议事厅讨论（现在会先给一屏摘要，再按需展开）
      - ⭐ 收藏高价值文章
      - 📝 写备注 / 🏷️ 打标签
      - 🕘 回看最近讨论
4. **沉淀档案** → 选择 `📚 文章库` → `⭐ 档案库`
   - 用 `🔎 设置筛选` 按关键词、标签、来源、时间搜索
   - 把值得反复讨论的文章留在个人档案库里
5. **向内探索** → 选择 `🧭 心智漫游`
   - `🪞 认识自己`：回答自我回顾问题
   - `🧠 哲思问题`：暴露自己的价值判断
   - `🧪 思想实验`：用设定和变体测试立场稳定性
   - 回答可以保存到长期记忆，之后在 `📜 回看我的回答` 中查看
6. **查看历史** → 选择 `🧠 认知回顾`

### 日常使用

**方式1：浏览模式（推荐）**

选择 `📚 文章库` → `📚 浏览文章`：
1. 从文章列表中选择感兴趣的文章
2. 先看导读了解核心内容
3. 需要时在浏览器中查看原文
4. 看到高价值内容时，顺手收藏、写备注、打标签
5. 准备好后发起讨论

**方式1.5：档案库复习**

选择 `📚 文章库` → `⭐ 档案库`：
1. 用关键词或标签快速找回之前收藏的高价值文章
2. 查看备注，想起当时为什么留下它
3. 回看最近讨论，把旧上下文接上继续思考

**方式2：心智漫游**

选择 `🧭 心智漫游`：
1. 从 `认识自己 / 哲思问题 / 思想实验` 中选一个入口，或直接随机一题
2. 先写下第一反应，不需要追求标准答案
3. 查看系统提炼出的核心线索、隐藏前提和继续追问
4. 选择是否保存到长期记忆

**方式3：一键流程**

直接选择 `🚀 今日练习`，一键完成：
- 抓取最新高质量内容
- 自动选择最佳文章
- 发起议事厅讨论
- 记录你的观点
- 生成认知对比报告

### 对话节奏更新

- **Council**：默认先给“结论 / 最大争议 / 新视角 / 最值得回应的问题”，你再决定要不要展开某个角色。
- **Your Turn**：不再直接丢给你空白输入框，而是先给几个起手句，降低开口成本。
- **Resolve**：智库模式下先给综合回应，再按需展开 Critic / Synthesizer / Mentor。

## 常见问题

### Q: 如何退出交互式菜单？
A: 选择 `❌ 退出`，或按 `Ctrl+C`

### Q: 如何使用传统命令行？
A: 直接运行命令，例如 `python -m src scout`

### Q: 如何检测 API 配置是否可用？
A: 在菜单里进入 `设置` → `检测 API 是否可用`。命令行也可以用：
`python -m src config --test --provider global`
检测 Scout 档：
`python -m src config --test --provider scout`
全部检测：
`python -m src config --test --provider all`

### Q: 想多看人文社科，少看科技文章怎么办？
A: 默认预设已经是 `humanities`。你也可以在 `.env` 里设置
`SCOUT_FEED_PRESET=humanities`
如果想临时切换，使用
`python -m src scout --preset humanities|mixed|tech`

### Q: 如何收藏高价值文章？
A: 在菜单里进入 `浏览文章`，选中文章后选择收藏。更推荐顺手补上备注和标签，这样后面更容易找回。命令行也可以用：
`python -m src favorite --item 1 --note "适合反复讨论现代性问题" --tags "history,modernity"`
查看档案库：
`python -m src favorites`
按标签筛选：
`python -m src favorites --tag history`
补充备注或标签：
`python -m src note --item 1 --text "和韦伯一起对读"`
`python -m src tag --item 1 --add "ethics"`

### Q: 旧文章会怎么清理？
A: 默认每次 Scout 后会清理 30 天前的普通文章；收藏文章、打过标签的文章、讨论过的文章、留下记忆的文章会保留。你可以先预览：
`python -m src cleanup --dry-run`

### Q: 如何恢复之前的对话？
A: 选择 `💬 Resolve` → `📜 查看并恢复历史会话`

### Q: 支持哪些 LLM？
A: 所有 OpenAI 兼容 API，包括：
- OpenAI (GPT-4, GPT-4o)
- DeepSeek
- Claude (通过兼容层)
- Gemini (通过兼容层)
- 本地 Ollama

### Q: 如何为不同任务配置不同模型？
A: 在配置时选择：
- **Global Default**：全局默认
- **Scout**：评分任务（推荐 DeepSeek，性价比高）
- **Council**：讨论任务（推荐 Claude/Gemini，逻辑强）
- **Memory**：认知分析（默认跟随 Council）

### Q: 遇到 "Your request was blocked" 错误怎么办？
A: 这是 API 的内容安全过滤机制触发。解决方案：
1. **切换模型**：某些模型对内容更敏感，尝试使用其他模型
2. **检查内容**：文章可能包含敏感话题
3. **使用 DeepSeek/OpenAI**：这些模型的内容策略相对宽松
4. **直接查看原文**：跳过导读，直接在浏览器中阅读
